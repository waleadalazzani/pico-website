Introducing Pico.

A stupidly simple & blazing fast, flat file cms.
Making the web easy.

Stupidly Simple

Pico makes creating and maintaining a website as simple as editing text files.

Markdown Formatting

Edit your website in your favorite text editor using simple Markdown formatting

Blazing Fast

Pico is seriously lightweight and doesn't use a database, making it super fast.

Twig Templates

Pico uses the Twig templating engine, for powerful and flexible themes

No Database

Pico is a " flat file " cms, meaning no database woe's.
no MySQL queries, nothing.

Open Source

Pico is completely free and open source, released under the mit license
